# Python Data Structures

## Overview

This repository contains Python programs demonstrating operations on lists, dictionaries, and collections. The exercises focus on data manipulation, filtering, searching, and aggregation using Python functions.

## Programs Included

### 1. Remove Duplicates

- Removes duplicate values from a list
- Returns a list containing unique elements

### 2. Remove Elements

- Removes specified elements from a list
- Demonstrates list manipulation techniques

### 3. Calculate Average Grades

- Stores subject grades in a dictionary
- Calculates the average grade across subjects

### 4. Find Common Elements

- Finds elements shared between two lists
- Demonstrates set operations and list processing

### 5. Filter Even Numbers

- Filters a list and returns only even numbers
- Demonstrates conditional processing and iteration
